# OLT Helper

OLT Helper é uma ferramenta executável desenvolvida em Python para automação de rotinas operacionais e diagnóstico de rede em ambientes GPON, com foco inicial em equipamentos da ZTE.

O projeto surgiu com o objetivo de reduzir o tempo gasto em verificações manuais realizadas diariamente no suporte técnico, centralizando comandos e automatizando consultas utilizadas no diagnóstico de ONU/ONTs.

A aplicação é executada via CMD/terminal, utilizando uma interface baseada em linha de comando para navegação entre as funcionalidades da ferramenta.

## Funcionalidades

* Consulta de informações de ONU/ONT
* Verificação de sinal óptico
* Consulta de scripts/configurações
* Execução de reset e reboot
* Remoção de provisionamento/script
* Automação de comandos via Telnet
* Padronização de diagnósticos operacionais

## Objetivo do projeto

O OLT Helper foi criado para auxiliar profissionais de suporte técnico e operações de rede, tornando processos repetitivos mais rápidos, organizados e eficientes.

Além da automação das rotinas, o projeto também serviu como aprendizado prático em:

* Python
* Automação de rede
* Arquitetura de software
* Estruturação de fluxos operacionais
* QA e validação de funcionalidades
* Integração via Telnet

## Segurança

Todas as demonstrações, prints e exemplos presentes neste repositório possuem informações sensíveis ocultadas, como:

* IPs
* VLANs
* Números de série
* Dados internos da infraestrutura

## Tecnologias utilizadas

* Python
* Telnetlib
* CMD/Terminal
* Automação CLI
* Redes GPON

## Observações

O projeto está em constante evolução e novas funcionalidades poderão ser adicionadas futuramente.

O repositório também contará com uma versão executável da aplicação, permitindo sua utilização sem necessidade de instalação prévia do Python no computador.
